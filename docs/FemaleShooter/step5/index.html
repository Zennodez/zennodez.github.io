    Female Shooter - User Guide

    Setting up the Animation System

    In this article we are going to prepare the Animation Blueprint for your character.
    This step is required whether you plan to use the custom animations that you've already made or reuse the included default animations that the template provides.

    There are 2 ways to do this,

    • You can Assign a Skeleton for your skeletal mesh,
    • Or you can Retarget the Animation Blueprint to your skeletal mesh (recommended)

    Assign a Skeleton

    • Right click on the skeletal mesh you wish to use for your character,
    • Choose Skeleton, then choose Assign Skeleton
    • Choose the GunnerGirl_Skeleton

    The GunnerGirl_Skeleton is based on the rescaled version of UE4 Mannequin skeleton, so any skeletal mesh based on the UE4 mannequin will most likely work with minimal issue.
    However, it's not really recommended to use this approach since it might not work with skeletal mesh whose joint orientation might be set differently from the UE4 Mannequin.

    Retarget the Animation Blueprint

    • Prepare your skeleton for retargeting See the retargeting preparation guide for detailed instructions.
    • Navigate to Content/FemaleShooter/Asset/GunnerGirl/Anim/GunnerGirlAnimation/GunnerGirl_AnimBP
    • Right click on the file, choose Retarget Anim Blueprints, then choose Duplicate Anim Blueprints and Retarget
    • Choose the skeleton asset of your skeletal mesh Try unchecking the Show Only Compatible Skeletons option if your skeleton asset is not listed on the option.
    • Add a prefix and assign the folder for the New Asset Name This step is optional, but recommended.
      It might be helpful when you need to locate the newly generated asset using its name.
    • Press Retarget

    This approach will work on most skeletal mesh, but it will generate a new Animation Blueprint for every skeletal mesh it retargets to.
    It might also create a copy of all the animations used in the Animation Blueprint. Starting from version 1.2.0, the Female Shooter template now store its animations inside a variable.
    Thus, by default, the retarget procedure will only generate a new Animation Blueprint and does not generate any extra animations.
    If the extra animations happened to be required for your project, you will need to explicitly perform the retargeting on those extra animations as well.

    Connecting the Animation Blueprint to your Character

    Once the Animation Blueprint is ready, we can now apply it into the character.
    Steps are as follows,

    • Open your character's blueprint
    • Choose the character's Mesh component
    • Go to Details tab, find Animation Mode, set its value to Use Animation Blueprint
    • Also on Details tab, find Anim Class, set its value to the AnimBP that you have prepared using the steps above.

    This should complete the default Female Shooter template setup for your character.

    If you're planning to use custom animations for your character, make sure to take a look at the next article about Using Custom Animations on the template.
